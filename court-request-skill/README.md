# Court Request Writer Skill — كاتب طلبات المحكمة

Claude Skill يساعد على صياغة طلبات المحكمة باللغة العربية بأسلوب مكتب الخبير الحسابي / أحمد عبدالرحيم الأنصاري (قيد رقم 200).

## المحتويات

```
court-request-skill/
├── SKILL.md                    ← التعليمات الرئيسية للـ Skill
├── README.md                   ← هذا الملف
└── references/
    ├── examples.md             ← 9 أمثلة حقيقية من المكتب
    ├── style-guide.md          ← قاموس العبارات والصياغات المعتمدة
    └── templates.md            ← قوالب جاهزة لكل نوع طلب
```

## كيفية الاستخدام

### الطريقة 1 — على Claude.ai (الأسهل)

1. ادخل على [claude.ai](https://claude.ai) وافتح إعدادات الـ **Skills** (Capabilities → Skills).
2. اضغط **Upload skill** أو **Create skill**.
3. ارفع مجلد `court-request-skill/` كاملاً (أو اضغطه ZIP وارفعه).
4. فعّل الـ Skill في أي محادثة جديدة.
5. اكتب: "عايز أعمل طلب محكمة" — وكلود هيسألك عن النوع والبيانات ويصيغهولك.

### الطريقة 2 — عن طريق Claude API

استخدم الـ Skill مع SDK الـ Skills الجديد من Anthropic عن طريق رفع المجلد كـ Skill resource، ثم تفعيله في طلب الـ API.

### الطريقة 3 — كمرجع يدوي

تقدر تفتح أي محادثة جديدة مع كلود وتلصق محتوى `SKILL.md` + `references/examples.md` كرسالة افتتاحية، وكلود يبدأ يساعدك مباشرة بنفس الأسلوب.

## مثال على الاستخدام

**أنت:** عايز أعمل طلب محكمة

**كلود:** أهلاً بك 🙏 أي نوع من الطلبات تريد إعداده اليوم؟
1. طلب تأجيل موعد تسليم تقرير الخبرة
2. طلب زيادة أتعاب الخبرة
3. طلب إزالة الدعوى من القضايا المستندة للخبير
4. ...

**أنت:** طلب تأجيل، الدعوى رقم 99 لسنة 2026 تجاري جزئي، الجهة محاكم دبي، تاريخ الحكم التمهيدي 01-04-2026، عايز التأجيل لـ 30-05-2026، السبب تأخر الأطراف في تقديم المستندات.

**كلود:** *(يصيغ الطلب كاملاً بنفس أسلوب المكتب)*

## ملاحظات مهمة

- التوقيع ثابت دائماً: `الخبير الحسابي / أحمد عبدالرحيم الأنصاري / قيد رقم 200`.
- الـ Skill مبني على 9 طلبات حقيقية من المكتب، فالأسلوب مطابق فعلياً وليس مقلداً.
- لو في نوع طلب جديد مش موجود في الأمثلة، الـ Skill هيستخدم نفس البنية والمفردات من `style-guide.md`.
